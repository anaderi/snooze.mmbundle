# snooze.mmbundle
MailMate Bundle to implement message snooze


## spark snooze

* later today + 3 hours
* this evening - 8pm
* tomorrow - 9am tomorrow
* tommorrow eve - 8pm
* weekend - sat 9am
* next week - 9am monday
* in a month - 9am
* pick a date

spark also moves to later imap folder
